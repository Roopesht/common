# Setup – FastAPI

## Objective
Run a FastAPI server with environment variable support.

---

## 1. Install FastAPI and Uvicorn

Make sure your virtual environment is active, then run:

```bash
pip install fastapi uvicorn python-dotenv
```

---

## 2. Create `main.py`

```python
from fastapi import FastAPI
import os
from dotenv import load_dotenv

load_dotenv()

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "API is running", "app": os.getenv("APP_NAME")}
```

---

## 3. Set up `.env` file

Create a file named `.env` in the same folder as `main.py`:

```
APP_NAME=GenAI App
```

> Never share or commit this file — it holds sensitive config values.

---

## 4. Access `.env` variables in code

```python
import os
app_name = os.getenv("APP_NAME")  # returns "GenAI App"
```

> `load_dotenv()` at the top of `main.py` loads the `.env` file automatically.

---

## 5. Run the server

**Mac/Linux:**
```bash
uvicorn main:app --reload
```

**Windows:**
```bash
python -m uvicorn main:app --reload
```

> `--reload` restarts the server automatically when you save a file.

**Stop the server:** Press `CTRL + C`

---

## Validation

Open these URLs in your browser:

- [http://127.0.0.1:8000](http://127.0.0.1:8000) — should return `{"message": "API is running", "app": "GenAI App"}`
- [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs) — interactive API explorer (auto-generated by FastAPI)
